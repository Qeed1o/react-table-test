Технические требования

Стек технологий:

React 18+.
TypeScript (строгая типизация).
Поддержка актуальной версии Google Chrome.

Дополнительные инструменты:

Стейт-менеджмент: Redux Toolkit.
UI-библиотека: Material-UI.


Дизайн

Визуальное соответствие: макету в Figma. Если будет необходимо - я отправлю тебе их в чат
Анимации: минималистичные (например, плавное появление модальных окон, уведомлений).

Функциональные требования
1. Форма входа

Валидация: обязательные поля (логин, пароль).
Обработка ошибок: вывод текста ошибки под полями или уведомление при ошибке API.
Логика запоминания:

Чекбокс "Запомнить меня" → токен в localStorage (сессия сохраняется).
Без чекбокса → токен в sessionStorage (сессия сбрасывается).

2. Список товаров

Структура: соответствие столбцам из Figma.
Пагинация: по 10 товаров на страницу.
Индикация загрузки: прогресс-бар при загрузке данных.
Сортировка: по столбцам (цена, рейтинг и т. д.), состояние сохраняется.
3. Добавление товара

Форма: открывается по кнопке "Добавить".
Поля: Наименование, цена, вендор, артикул.
Уведомление: Toast при успешном добавлении.
(API-сохранение не требуется.)
4. Логика интерфейса

Подсветка рейтинга: красный цвет, если рейтинг < 3.
Поиск: через API (по названию, артикулу и т. д.).
